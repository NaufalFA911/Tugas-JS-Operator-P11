### Deskripsi Program
Program ini adalah aplikasi **Simulasi Kasir Otomatis** sederhana berbasis web. Program ini dirancang untuk mengintegrasikan komponen HTML sebagai struktur halaman, CSS internal untuk desain antarmuka, dan JavaScript sebagai pengolah logika data.

Secara spesifik, program ini menerapkan materi inti JavaScript sebagai berikut:
1. **Variabel:** Menggunakan `let` untuk menyimpan data harga barang, jumlah pembelian, nominal diskon, dan total belanja.
2. **Operator Aritmatika (`*`):** Menghitung total belanja awal (harga dikali jumlah) dan mengalikan persentase untuk mencari nilai diskon.
3. **Operator Pembanding (`>`):** Memeriksa kondisi kelayakan diskon. Jika total belanja di atas Rp 100.000, pelanggan berhak mendapatkan diskon 10%.
4. **Operator Penugasan (`-=`):** Memotong total belanja awal dengan nominal diskon secara ringkas (`totalBelanja -= diskon`).
5. **Output:** Menampilkan hasil kalkulasi JavaScript langsung ke elemen halaman HTML menggunakan `document.getElementById().innerText`.
